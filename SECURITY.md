### CRIT-001: API Keys Hardcoded in Source Code

| File | Key Type | Status |
|------|----------|--------|
| `solana.ts` | Helius RPC API Key | ⚠️ ROTATE IMMEDIATELY |
| `routes.ts` | Alchemy API Key | ⚠️ ROTATE IMMEDIATELY |

**Fix**:
```typescript
// Before (dangerous):
const HELIUS_API_KEY = "a24bbb32-...";

// After (safe):
const HELIUS_API_KEY = process.env.HELIUS_API_KEY;
if (!HELIUS_API_KEY) throw new Error('HELIUS_API_KEY env var required');
```

### CRIT-002: Private Keys Returned by Unauthenticated Endpoint

`GET /api/empire/wallets` returns raw EVM private keys in plaintext with no authentication.

**Immediate action**: Block this endpoint or add auth middleware. Never return `privateKey` in API responses.

---

### HIGH-001: Ralph Agent Control Endpoints Unauthenticated

`POST /api/ralph/start|stop|strategy` — no auth required. Critical once real swap execution is enabled.

**Fix**: Add `Authorization: Bearer <RALPH_API_KEY>` middleware before all Ralph routes.

---

### MEDIUM-001: EmpireSpawner Ephemeral Keypairs

Keypairs generated on every constructor call are lost on restart. Funded addresses become inaccessible.

**Fix**: Persist keypair to encrypted local file or env var on first generate; load on subsequent starts.

## Environment Variables Required

Create a `.env` file (never commit it):

```env
# Helius RPC
HELIUS_API_KEY=your-helius-key-here

# Alchemy
ALCHEMY_API_KEY=your-alchemy-key-here

# Ralph agent API auth
RALPH_API_KEY=$(node -e "console.log(require('crypto').randomBytes(32).toString('hex'))")

# EmpireSpawner persistent keypair (base58)
SPAWNER_PRIVATE_KEY=your-spawner-private-key

# JWT secret for API auth
JWT_SECRET=$(node -e "console.log(require('crypto').randomBytes(64).toString('hex'))")
```

Add to `.gitignore`:
```
.env
.env.local
*.keypair.json
.spawner-keypair.json
```
